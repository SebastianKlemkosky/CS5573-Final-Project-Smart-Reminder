About Smart Reminder

What is Smart Reminder?

Smart Reminder is a cloud-based web application designed to help users create and manage daily reminders for important activities such as health, fitness, work, relationships, and other personal tasks.

Project Purpose

The purpose of this project is to build a simple and organized reminder system through a web interface that users can access online.

Features

Create reminders, view saved reminders, organize reminders by category, and manage daily tasks in one place.

Technologies Used

Python, Flask, HTML/CSS, and PythonAnywhere.
